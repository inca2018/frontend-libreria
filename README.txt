# Instrucciones para levantar el proyecto Angular

## Requisitos Mínimos

Antes de comenzar, asegúrate de tener los siguientes requisitos instalados:

1. **Node.js**: Se recomienda la versión LTS de Node.js.
   - [Descargar Node.js](https://nodejs.org/)

2. **npm**: Se instala automáticamente con Node.js.

3. **Angular CLI**: Instala Angular CLI globalmente usando el siguiente comando:
   ```bash
   npm install -g @angular/cli

## Levantar Frontend Angular 

En la ubicacion del proyecto ejecutar el codigo para iniciar.
- ng serve --no-hmr
